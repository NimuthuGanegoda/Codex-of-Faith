The **Codex of Faith** is a curated, open-source archive documenting the foundational beliefs, histories, practices, and stories of the world's major religions. It includes a modern, responsive web application built with React and TypeScript to gracefully present these sacred texts.

## 📖 Contents

This repository houses comprehensive summaries of:

- **[Buddhism](Buddhism.md)** - The Path of the Awakened One
- **[Hinduism](Hinduism.md)** - Sanatana Dharma (The Eternal Way)
- **[Christianity](Christianity.md)** - The Faith of the Risen Christ
- **[Islam](Islam.md)** - Submission to the One God
- **[Judaism](Judaism.md)** - The Covenant of the People
- **[Sikhism](Sikhism.md)** - The Path of Equality and Service

*Sacred stories and parables are integrated directly into the web interface.*

## 🚀 The Web Application

A sleek, responsive frontend serves the archive to the public. It is built using:
- **React & TypeScript** for robust, scalable UI development.
- **Vite** for fast, highly optimized builds.
- **React Markdown** for dynamic, styled text rendering from the core archive.

### Local Development

To run the application locally, ensure you have Node.js (v18+) installed.

1. Navigate to the web directory:
   ```bash
   cd codex-web
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
